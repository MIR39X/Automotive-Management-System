# AMS Project

## Overview
AMS (Automotive Management System) is a simple web-based application designed to manage vehicles, customers, sales, and maintenance records. This project is built using PHP and MySQL.

## Features
- **Vehicle Management**: Add, edit, view, and delete vehicles.
- **Customer Management**: Manage customer details.
- **Sales Tracking**: Record and track sales.
- **Maintenance Records**: Log maintenance activities.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/MIR39X/ams_Project.git
   ```
2. Set up a local web server (e.g., XAMPP).
3. Import the database schema from `db.sql`.
4. Update the database connection settings in `includes/db.php`.

## Usage
- Navigate to the home page to view available vehicles.
- Use the navigation bar to access different modules (Vehicles, Customers, Sales, Maintenance).
